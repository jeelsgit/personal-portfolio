# Jeel Tandel — Portfolio

Personal portfolio site built with React, TypeScript, Vite, and Tailwind CSS.

**Live:** [jeeltandel.com](https://www.jeeltandel.com/)

![Portfolio screenshot](./screenshot.png)

## About

I'm a Software Engineer based in Ottawa, Canada, working in Python (FastAPI), Java, and PostgreSQL — backend APIs, ML/NLP services, and data pipelines. This site collects my experience, projects, and certifications.

## Tech Stack

- **Framework:** React 19 + TypeScript
- **Build:** Vite
- **Styling:** Tailwind CSS
- **Fonts/Icons:** Inter, JetBrains Mono, Font Awesome

## Features

- Fully responsive, mobile-first layout
- Accessible navigation (skip links, ARIA labels, visible focus states)
- Animated hero with SVG globe and terminal components
- Content driven from a single typed `constants.ts` for easy updates

## Run Locally

**Prerequisites:** Node.js 18+

```bash
npm install
npm run dev
```
